# Neovim

## Installation

```bash
git clone --recurse-submodules git@github.com:vgitv/nvim.git ~/.config/nvim
```

Or

```bash
git clone git@github.com:vgitv/nvim.git ~/.config/nvim
```

```bash
git submodule update --init
```


## Update submodules

```bash
git submodule update --remote --merge
```
